# Data

Download the dataset from Kaggle and place both files in this folder:

**Dataset:** [UK Road Safety Data — Accidents and Vehicles](https://www.kaggle.com/datasets/tsiaras/uk-road-safety-accidents-and-vehicles)

| File | Size |
|------|------|
| `Accident_Information.csv` | ~600 MB |
| `Vehicle_Information.csv` | ~500 MB |

Both files are excluded from git via `.gitignore` due to their size.
